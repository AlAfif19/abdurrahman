# Al Afif Abdurrahman Portfolio

Split Personality Portfolio Website for Al Afif Abdurrahman.

## Run Locally

```bash
python run.py
```

The runner installs frontend dependencies when `frontend/node_modules` is missing, then starts the Vite development server.

## Frontend Commands

```bash
cd frontend
npm run test -- --run
npm run build
npm run dev
```

## Project Shape

- `frontend/`: React, Vite, TailwindCSS portfolio app.
- `backend/`: minimal FastAPI-compatible placeholder for future backend work.
- `foto/`: original source photos.
- `docs/superpowers/`: design and implementation planning docs.
